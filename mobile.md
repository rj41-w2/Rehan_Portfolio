Mobile View UI Enhancements
This document outlines the required adjustments for the Mobile View of the portfolio website. These changes must be applied using media queries or responsive utility classes to ensure the Desktop View remains unchanged.

1. Header & Navigation (Mobile Only)
Menu Toggle: Replace the standard horizontal navigation links with a three-dot (vertical ellipsis) menu icon positioned on the top-right side.

Functionality: Clicking the three-dot icon should trigger a dropdown or a slide-out drawer containing the navigation links.

Theme Toggle: Keep the Theme Toggle button visible and positioned immediately next to the three-dot menu icon.

Guestbook Integration: Move the Guestbook icon/link from the main UI and nest it inside the mobile navigation menu (the list revealed by the three-dot icon).

2. Footer Adjustments (Mobile Only)
Alignment: Change the alignment of the copyright text (© 2026 Rehan Jamil. All rights reserved.) to the right side of the footer.

Typography: Reduce the font size significantly. The text should be small and subtle—just clear enough to be legible without drawing focus.
